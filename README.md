# 周线量变爆发选股 V2

A股周线级别量化选股策略，每个交易日收盘后自动更新结果并发布到 GitHub Pages。

## 策略逻辑

三个条件**同时满足**才会入选：

| 条件 | 说明 |
|------|------|
| **BOLL 扩张** | 布林带上轨上升、中轨上升、下轨下降（开口扩张） |
| **放量（双重验证）** | ① 52周内任意一周成交量 > 前一周 **3倍**；② 最近26周内任意一周成交量 > 前一周 **1.5倍** |
| **MACD 零上金叉** | 26周内任意一周，DIF 上穿 DEA 且当周 DEA > 0 |

> 放量条件①②须**同时满足**。

## 自动更新

每个交易日（周一至周五）北京时间 **16:30** 自动运行策略，结果自动推送到 GitHub Pages。也可在仓库 Actions 页面手动触发。

## 快速部署

1. **Fork** 本仓库到你的 GitHub 账号
2. 进入仓库 **Settings → Pages**，将 Source 设置为 **GitHub Actions**
3. 在 **Actions** 页面手动触发一次 `workflow_dispatch` 以初始化页面
4. 之后每个交易日 16:30 后自动更新，访问 `https://<你的用户名>.github.io/<仓库名>/` 查看结果

## 数据说明

- 数据源：腾讯财经（无需 API Key，国内外均可访问）
- 覆盖范围：沪深主板、中小板、创业板、科创板
- 自动过滤 ST、退市、停牌股票
